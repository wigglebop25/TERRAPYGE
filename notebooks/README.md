# Jupyter Notebooks

This directory contains all Jupyter notebooks for the TERRAPYGE project.

## Available Notebooks

1. **01_explore_dem.ipynb** - DEM exploration: loads raw SRTM DEM, plots elevation, computes basic statistics, renders 3D surface.
2. **02_preprocess_dem.ipynb** - DEM conditioning with WhiteboxTools: fills depressions and pits, derives slope and aspect. **Production notebook.**
3. **02_preprocess_dem_pure_python_BASELINE.ipynb** - DEM conditioning using pure Python (no GDAL/WhiteboxTools). **BASELINE ONLY** - slope is computed with degree-spaced pixels (physically incorrect). Retained for ablation/comparison.
4. **03_slope_units_extraction_fixed.ipynb** - Slope unit extraction using `r.slopeunits` (GRASS GIS). Requires GRASS GIS installed and on PATH.

## Notebook Execution Order

1. `01_explore_dem.ipynb` (load and inspect DEM)
2. `02_preprocess_dem.ipynb` (condition DEM with WhiteboxTools)
3. `03_slope_units_extraction_fixed.ipynb` (extract slope units via r.slopeunits)

Notebooks 04-07 (graph construction, GNN modeling, evaluation, visualization) are not yet created.

## Prerequisites

- Python environment activated (`venv` at `D:\TERRAPYGE\venv`)
- DEM at `data/raw/dem/CebuCity_DEM.tif` (from Google Earth Engine)
- WhiteboxTools (`pip install whitebox`)
- GRASS GIS (for notebook 03 only) - install from https://grass.osgeo.org/download/windows/
- r.slopeunits add-on: `grass --tmp-mapset --exec g.extension extension=r.slopeunits`

## Standalone Scripts

Equivalent standalone scripts exist at the project root for headless execution:

- `phase1_dem_preprocessing.py` - DEM preprocessing pipeline (reproject, mask, condition, slope/aspect)
- `phase1_slope_units.py` - Slope unit extraction via r.slopeunits

Run with: `python phase1_dem_preprocessing.py` / `python phase1_slope_units.py`

## Data Dependencies

- DEM from Google Earth Engine (SRTM 30m) - downloaded to `data/raw/dem/`
- Conditioned DEM output to `data/processed/` (generated by notebook 02)
- Slope units output to `data/processed/slope_units.shp` (generated by notebook 03)
- Philippine Soil Series - available at `Philippine Soil Series (shapefile)/`
